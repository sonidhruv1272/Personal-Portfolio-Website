## Projects

The Projects section showcases engineering and software projects that demonstrate my experience designing, building, and testing real-world systems.

Featured projects include:

### Autonomous Sumobot Robot

An autonomous competition robot developed using Arduino and embedded C++. The system integrates ultrasonic and infrared sensors for object detection and boundary detection while using optimized motor control algorithms for real-time decision making.

### Smart Doorbell System (ESP32-CAM + Blynk IoT)

An IoT-based smart doorbell system using ESP32-CAM, embedded firmware, and the Blynk platform. The project enables real-time Wi-Fi notifications, remote monitoring, and mobile device integration.

### Fingerprint Door Lock System

A biometric security system developed using Arduino and a fingerprint sensor. The system implements embedded C++ authentication logic, servo-based locking control, and secure access validation.

### PCB Design Project

A hardware design project focused on creating optimized PCB layouts using CAD tools. The project involved component placement, trace routing, circuit simulation, and power distribution optimization.

### Personal Portfolio Website

A responsive portfolio website developed and deployed using GitHub Pages. The website showcases my engineering projects, technical skills, resume, and professional experience.

Each project demonstrates my ability to combine software, hardware, and engineering principles to develop practical solutions.
